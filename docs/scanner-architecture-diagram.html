Layer 1 — API

Next.js API Routes

POST /api/scans/create
Creates scan record, calls initializeProgress() which auto-detects engine (Nmap or builtin). Stores scan config + engine type in DB.
POST /api/scans/[id]/execute
UI polls this endpoint. Each call runs one batch of checks via executeNextBatch(). Returns progress JSON until status = "completed".
